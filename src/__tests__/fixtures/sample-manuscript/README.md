# Sample Manuscript: "Building Web Applications"

This is a test manuscript for the Writer's Aid MCP tool. It intentionally contains various issues to test the quality validation features.

## Intentional Issues

This manuscript contains the following intentional issues for testing:

1. **Structure Issues:**
   - Skipped heading levels (H1 to H3)
   - Duplicate headings
   - Deep nesting (H5+)

2. **Terminology Inconsistencies:**
   - "email" vs "e-mail" vs "Email"
   - "API" vs "api" vs "Api"
   - "database" vs "data-base"

3. **Quality Issues:**
   - TODO and FIXME markers
   - Broken internal links
   - Duplicate content

4. **Style Issues:**
   - Inconsistent list markers
   - Mixed heading styles
   - Very long sentences (readability)

## Structure

- `/chapters/` - Main content chapters
- `/appendix/` - Supplementary materials
